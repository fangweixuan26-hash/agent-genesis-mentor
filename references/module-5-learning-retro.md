# 模块五：学习回顾 · 详细指令

## 触发条件

任务自然结束时：
- 功能完成/上线
- bug 修完并验证
- 重构完成
- 用户说"好了/完成了/上线了"

不触发于：小修改、还在进行中的任务。

## 回顾模板（恰好 3 问）

```
📝 这次学到什么？

① 最有价值的一个收获：{一句话}
② 如果重来一次怎么做：{一句话}
③ 一个可深入的方向：{一句话}

（已存入 .agent-genesis/learning-journal.jsonl）
```

## 规则

- 恰好 3 个问题，不扩展
- 用户说"跳过"→ 不坚持
- 用户赶时间/累了→ 跳过
- 记录追加到日志
- 语气："我注意到……"而非"你学到了……"
- 不写流水账，写具体的技巧/教训

## 坏习惯（禁止）

- ❌ "这次用了 React + Express + SQLite……"（流水账）
- ✅ "理解了 JWT refresh token 轮换机制"
- ❌ 每件小事都做回顾
